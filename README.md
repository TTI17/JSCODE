# Расчёт потенциальной энергии тела

## Описание проекта
Простой калькулятор, позволяющий пользователю вычислять потенциальную энергию тела по заданным массе и высоте.

---

## Цель
### Разработать программу для определения потенциальной энергии, используя ввод на клавиатуре массу и высоту тела пользователем 

## Функциональность
- Ввод массы и высоты через форму на веб-странице.  
- Расчёт потенциальной энергии по формуле:

- `E = m * g * h`, где `g = 9.8` — ускорение свободного падения.  
- Проверка корректности данных:
- масса и высота не могут быть отрицательными;
- при ошибке выводится сообщение пользователю.
- Отображение результата на странице.

---

## Инвариант
- Масса и высота — положительные числа.  
- Гравитационная постоянная `G = 9.8` неизменна.

---

## Концепция
Пользователь вводит исходные данные (массу и высоту) →  
JavaScript-функция обрабатывает значения →  
результат выводится на страницу.  

---
## Требования к программе
1. Корректный ввод данных пользователем.  
2. Отсутствие ошибок при отрицательных или некорректных значениях.  
3. Понятный и минималистичный интерфейс.  
4. Код оформлен по стандартам **CodeStyle JS**.
